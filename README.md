# Blog 

### 📒 개요
스마일게이트 윈터 데브 캠프 1인 프로젝트 blog 입니다. 기능 구현에 초점을 맞추었습니다. 한달 동안 개발한 후에 배포하여 개인 기술 블로그로 사용할 계획입니다.
### 📗 기간
2022.12.05. ~ 2022.12.24.
### 📙 Requirements
- 글 쓰기/ 수정
- 글 목록/삭제
- 댓글
- 관리자 도구
- 좋아요
- RSS

### 📘 Tech stack
#### front
- React   

#### back
- node.js  v16.18.1
- MySQL    v5.7


#### 개발환경
- VScode
